# Lista de Tarefas 📝

Este é um projeto de Lista de Tarefas desenvolvido com **Node.js**, **React (JSX)** e **PostgreSQL**. A aplicação permite gerenciar tarefas de forma simples e eficiente, com funcionalidades de criação, edição, exclusão e marcação de tarefas concluídas.

---

## 🛠️ Funcionalidades

- Adicionar novas tarefas
- Editar tarefas existentes
- Excluir tarefas
- Marcar tarefas como concluídas
- Interface responsiva e moderna desenvolvida com React (JSX)

---

## 🚀 Tecnologias Utilizadas

- **Front-end**: React (JSX, Hooks, Context API)
- **Back-end**: Node.js (Express)
- **Banco de Dados**: PostgreSQL
- **Outras Ferramentas**: Axios

---

## 📋 Pré-requisitos

Antes de começar, certifique-se de ter as seguintes ferramentas instaladas:

- [Node.js](https://nodejs.org/)
- [PostgreSQL](https://www.postgresql.org/)

## 🚀 Como Rodar o Projeto

### Configuração do Banco de Dados

1. Atualize as credenciais de acesso ao banco de dados no arquivo `.env`:

   ```env
   DB_HOST=localhost
   DB_USER=seu_usuario
   DB_PASSWORD=sua_senha
   DB_NAME=lista_tarefas
   DB_PORT=5432

2. Crie um banco de dados no PostgreSQL:
   ```sql
   CREATE DATABASE lista_tarefas;
